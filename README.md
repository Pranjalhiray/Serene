## 🌸 About

**Serene** is a beautiful, dark-themed task manager built with pure HTML, CSS, and JavaScript. No frameworks, no dependencies — just clean, fast, and fun task management with a stunning purple aesthetic.

---

## ✨ Features

- 🔥 **Add Tasks** — with priority levels (High / Medium / Low)
- 📅 **Due Dates** — with smart labels (Due Today, Tomorrow, Overdue)
- ✅ **Mark Complete** — with confetti celebration 🎉
- 🗑️ **Delete Tasks** — smooth slide-out animation
- 🔍 **Filter Tasks** — All, Pending, Done, High Priority, Overdue
- 📊 **Progress Bar** — visual completion tracker
- 💾 **localStorage** — tasks saved automatically, survive page refresh
- 💜 **Mood Messages** — random motivational quotes
- 🌟 **Toast Notifications** — friendly feedback on every action
- ✨ **Particle Effects** — sparkles when adding tasks
- 🎊 **Confetti** — celebration when you complete a task
- 📱 **Fully Responsive** — works on mobile & desktop

## 💻 Run Locally

```bash
# 1. Clone the repo
git clone https://github.com/YOUR_USERNAME/serene.git

# 2. Open in browser
cd serene
open index.html
```

That's it! No `npm install`, no setup needed. 🎉

## 🌐 Deploy to Vercel

### Option 1 — Vercel Dashboard (easiest)
1. Push this repo to GitHub
2. Go to [vercel.com](https://vercel.com) → **Add New Project**
3. Select your `serene` repo
4. Click **Deploy** 🚀

### Option 2 — Vercel CLI
```bash
npm i -g vercel
vercel
```

## 📦 Deploy Updates

Every time you push to GitHub, Vercel auto-redeploys:

```bash

```
↓ Vercel detects the push and goes live automatically ✨
